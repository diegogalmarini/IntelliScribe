# Subir Archivos

Si ya tienes una grabación realizada con otro dispositivo (móvil, grabadora externa, vídeo descargado), puedes subirla a Diktalo para su procesamiento.

### Formatos Soportados
*   **Audio:** .mp3, .wav, .m4a, .ogg, .aac, .flac
*   **Video:** .mp4, .mov, .avi, .webm (extraeremos el audio automáticamente)

### Pasos
1.  Ve a **"+ Nueva Sesión"** > Pestaña **"Subir"**.
2.  Arrastra tu archivo al área designada o haz clic para explorar.
3.  Selecciona el idioma original del audio (o déjalo en Auto).
4.  Pulsa **"Transcribir"**.

El archivo se procesará en segundo plano y recibirás una notificación cuando esté listo.
