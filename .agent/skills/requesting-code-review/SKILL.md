# Requesting Code Review

## Overview
Get structured feedback on your changes before considering a task complete.

**Core principle:** Early feedback prevents wasted work.

## The Process

### 1. Prepare Changes
Ensure your code is clean, documented, and tests pass.

### 2. Identify Scope
What exactly did you change? What should the reviewer focus on?

### 3. Request Review
Announce: "I'm requesting a code review using superpowers:requesting-code-review."

### 4. Use the Reviewer Template
