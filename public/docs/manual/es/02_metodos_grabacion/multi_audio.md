# Multi-Audio y Asignación de Oradores

El modo **Multi-Audio** es una función avanzada diseñada para escenarios de grabación profesional donde cada participante tiene su propio micrófono dedicado, generando archivos de audio separados.

![Multi-Audio](/docs/screenshots/es/upload/02_multiaudio.png)

---

## ¿Por qué usar Multi-Audio?

En una grabación de una sola pista (mono o estéreo mix), si dos personas hablan al mismo tiempo, las voces se superponen y la transcripción puede sufrir. Con **Multi-Audio**, Diktalo recibe cada voz en un canal limpio.

*   **Diarización Perfecta:** La IA sabe exactamente quién habla porque cada archivo corresponde a un orador único.
*   **Mejor Calidad:** Elimina el ruido de fondo de otros micrófonos cuando una persona no está hablando.

---

## Cómo configurar una sesión Multi-Audio

1.  Graba tu sesión usando una grabadora multicanal o interfaces de audio separadas.
2.  En Diktalo, ve a **Subir Archivo**.
3.  Activa el switch **"Modo Multi-Pista"**.
4.  Sube los archivos individuales (ej. `Juan.mp3`, `Maria.mp3`).
5.  **Asigna las etiquetas:** El sistema te pedirá que escribas el nombre del orador para cada archivo subido.
6.  Procesa la sesión.

El resultado será una única transcripción combinada con una precisión de identificación de oradores del 100%.
